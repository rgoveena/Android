Creating the project: cordova create MediaCaptureAPIDemo com.ucsc.mediaCaptureAPIDemo MediaCaptureAPIDemo
Got to the project folder : cd MediaCaptureAPIDemo/
Adding the platform: cordova platform add android
Adding the plugin :  
				1) cordova plugin add org.apache.cordova.media-capture
				2) cordova plugin add org.apache.cordova.console
				3) cordova plugin add org.apache.cordova.dialogs
Open the Webstorm and open the project's www folder.
