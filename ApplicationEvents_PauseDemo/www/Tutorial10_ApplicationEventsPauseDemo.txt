Creating the project:  cordova create ApplicationEvents_PauseDemo com.ucsc.applicationEvents_PauseDemo ApplicationEvents_PauseDemo
Got to the project folder : cd ApplicationEvents_PauseDemo/
Adding the platform: cordova platform add android
Adding the plugin :  
				1) cordova plugin add org.apache.cordova.console
				2) cordova plugin add org.apache.cordova.dialogs
Open the Webstorm and open the project's www folder.
